# geometry-dash-clone
This is the structured version of our OOP Project, a simple Geometry Dash Clone.
It is built using SFML 3.0.2 for VC++ 17 for the 64-bit Windows platform
The Version used for SFML is the one for the Visual Studio compiler (MSVC)

# run-requirements
To open the project with all the required settings, Visual Studio 2022 or later.
To run any build or exe, the .dll files uploaded in the repository are required.
The .dll files must be in the same directory as the exe. If we build using visual studio, they must be in the same directory
as the solution and the main.cpp file.

# External
The External folder contains the SFML library required to build and run the project.

# Assets
There are three folders within the Assets folder.

1: The Fonts folder contains all the fonts used in the game, in .ttf format.
2: The Sounds folder contains all the sound effects and music used in the game, in .wav format.
3: Sprites contains the images used in the game as well as any unused sprites, in .png format.
